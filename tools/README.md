# Tools Modules

This folder contains manager scripts and panels for advanced editor tools:

- ContentBrowserPanel: Asset management
- EditorUtilityWidgetPanel: Custom editor actions
- WorldPartitionManager: Streaming and chunking
- LandscapeManager: Terrain sculpting
- FoliageTool: Foliage painting
- ModelingModeManager: Mesh editing
- GeometryScriptingManager: Procedural geometry
- PCGManager: Procedural content generation
- WaterSystemManager: Water tools
- WorldMetricsManager: LOD and metrics

## Usage

Integrate these tools into your editor for enhanced world building and asset workflows.
