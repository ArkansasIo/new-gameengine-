# Animation Modules

This folder contains scripts and panels for animation and cinematics tools:

- SkeletalMeshEditor: Edit skeletal meshes and animation blueprints
- ControlRigManager: Procedural rigging and animation
- IKRigManager: IK rig and retargeting pipelines
- Sequencer: Cinematics and timeline editor
- TakeRecorder: Capture performance and animation
