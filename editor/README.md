# Editor Modules

This folder contains UI panels and manager scripts for core editor features:

- WorldOutlinerPanel: Displays scene actors
- DetailsPanel: Shows actor properties
- ModesPanel: Switches between editor modes
- SettingsPanel: Configures world and project settings

## Usage

Add these panels to your main editor scene for a modular editor interface.

Each panel is implemented as a Godot scene (.tscn) and script (.gd).
